# Model Weights Folder

Copy all required model weights into this folder before starting the backend.

This repository does not store the actual `.pt` and `.pth` weight files in Git.
Get them from the shared Google Drive and place them here.

Expected examples:

- `YOLOv11_d1.pt`
- `YOLOv11_d2.pt`
- `YOLOv11_d3.pt`
- `YOLOv12_d1.pt`
- `YOLOv12_d2.pt`
- `YOLOv12_d3.pt`
- `YOLOv26_d1.pt`
- `YOLOv26_d2.pt`
- `YOLOv26_d3.pt`
- `SSD_RESNET50_d1.pth`
- `SSD_RESNET50_d2.pth`
- `SSD_RESNET50_d3.pth`

Do not rename the files unless you also update `backend/app/config.py`.
